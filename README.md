# Install Remotly

## Install using curl

```sh
sh -c "`curl -fsSL https://raw.githubusercontent.com/michaweber/mydotfiles/master/setup.sh`"
```

## Install using wget
```sh
wget -O - https://raw.githubusercontent.com/michaweber/mydotfiles/master/setup.sh | sh
```
